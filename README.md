# Tour Booking Manager

Веб-приложение для бронирования туров, отелей и экскурсий. Позволяет пользователям искать и оформлять поездки, а администраторам и менеджерам управлять каталогом услуг.

Тур, в данном проекте, это совокупность места проживания и подготовленных экскурссий
## Функционал

### Пользователи
* Регистрация, авторизация и личный кабинет (смена email/пароля).
* Просмотр каталога туров, отелей и экскурсий.
* Детальная страница тура с привязанным отелем и расписанием.
* Корзина и история оформленных заказов.
* Переключение валют в интерфейсе.

### Администраторы и менеджеры
* Панель управления (CRUD) для туров, отелей и экскурсий.
* Конструктор туров: привязка отеля и составление графика экскурсий.
* Управление пользователями и их ролями (только для `ADMIN`).

## Структура URL (Эндпоинты)

* `/auth/register`, `/auth/login` - авторизация
* `/users/me` - профиль пользователя
* `/tours` - каталог туров
* `/hotels` - список отелей
* `/excursions` - список экскурсий
* `/orders/cart` - корзина
* `/orders/history` - история заказов

## Ролевая модель

* `USER` - поиск туров, покупка, просмотр истории заказов.
* `MANAGER` - наполнение каталога (туры, отели, экскурсии).
* `ADMIN` - полный доступ к каталогу и управление пользователями.
